# PDF2PNG

自动转换 xcassets 里面的 pdf 文件为 png 文件，同时修改 Contents.json 文件，目的就是 Xcode 工程不用 pdf 矢量图，相信很多人会面对瘦身需求，希望这个工具对你有用。

### Usage

需要安装 Ghostscript，brew install gs
使用示例 ./pdf2png.sh xx.xcassets
